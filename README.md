Bank Management System

1. Project Description
This is a simple Bank Management System developed using Java Servlets, JSP, and HTML.  
The project uses Java File Management instead of a database to store and manage data.

 2. Technologies Used
- Java (Servlets)
- JSP (Java Server Pages)
- HTML & CSS
- Java File Handling
- Apache Tomcat 9 Server
3. Storage Technique
All user details, account information, and transaction data are stored using Java file handling concepts such as File, FileWriter, and FileReader.

4. Modules Implemented
- User Registration
- User Login
- Account Creation
- Balance Enquiry
- Deposit and Withdrawal

 5. Functionalities
The system allows users to create bank accounts, log in securely, perform basic banking operations, and store all records in files.

 6. Architecture
The project follows MVC architecture:
- JSP handles the View
- Servlets act as the Controller
- File system acts as the Model

  
 7. Server Used
Apache Tomcat 9 is used to deploy and run the web application.

 8. How to Run the Project
1. Import the project into Eclipse IDE  
2. Configure Apache Tomcat 9 server  
3. Ensure file paths are correctly set  
4. Run the project on the server  
5. Access the application through browser

 9. Purpose of the Project
This project is developed to understand Java Servlets, JSP, and file management concepts without using a database.
